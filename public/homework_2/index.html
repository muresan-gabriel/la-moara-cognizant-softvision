First problem.

Create a function which takes a number as a parameter.
If the number is even, the function returns the number substracted by two,
otherwise, if it's odd, the function returns the double of the number.

script.js
function isEvenOrOdd(number) {
  if (number % 2 === 0) {
      return (number -= 2);
  }
  return number * 2;
}

console.log(isEvenOrOdd(5));
console
10
